 1 уровень сложности: 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
1.для начала нам нужно в гит зайти в рабочюю директорию.Потом надо зайти на github зайти на главную страницу репозитороия.Над списков файлов щёлкнуть на Code затем скопировать адресс ссылки, 
и щёлкнуть SSH.Потом нужно склонировать в гит с кодом к примеру: $ git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY.и нажать ввод что бы создать локальный клон.
2 Как получить на локальный комьютер изменения с github.com
2. с помощью команды git fetch.
3.Какую информацию нам дают команда :
git status - проверка статуса
git log - просмотр изменений 
git remote -v - просмотреть список настроенных удалённых репозиториев,и ключ -v, чтобы просмотреть адреса для чтения и записи, привязанные к репозиторию
